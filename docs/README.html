Setup

Prerequisites

Getting started with Playbook depends on setting up a new Hubot, then just add the Playbook library…

You will need node.js and npm. Once those are installed, we can install the hubot generator and dev tools used by Playbook:

Globals: npm install -g coffee-script gulp nodemon yo generator-hubot

Follow the rest of the getting started with hubot steps.

Add Playbook to your bot: npm install --save hubot-playbook

Look at [examples] and [docs] for how to use Playbook and it’s modules to add conversational logic to your bots.
